Путь клиента: 
1.Пользователь открывает PWA (или сайт), видит лаконичную форму. Минимум полей: Email и Пароль. После входа — мгновенный редирект в меню.

2.Пользователь видит цену и кнопку «Добавить». Система лояльности (бонусы) всегда перед глазами — это стимулирует добавить еще одну позицию.

3.Нажатие на кнопку корзины открывает список. Здесь происходит удаление лишнего или подтверждение заказа.

4.После заказа пользователь переходит в профиль, где видит статус: Новый → Готовится → Готов.

5.Когда статус меняется на Готов, клиент идет к стойке.
